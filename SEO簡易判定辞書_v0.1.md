# お客様どっと混む SEO簡易判定辞書 v0.1

作成日: 2026-05-11  
目的: 無料診断・β版診断で、地域店舗のSEO/MEO/GEOを簡易判定するための基準語彙と確認項目を定義する。

---

## 1. 基本方針

SEO/GEO診断は、公式サイト、Instagram、YouTube、TikTok、X、Google Mapsなど、判定対象となるページが存在する場合だけ行う。

存在しない媒体は **未判定** とし、原則として総合点の直接減点にしない。ただし、業種や集客目的上、存在しないこと自体が機会損失になる場合は「改善提案」として扱う。

例:

- 公式サイトなし: SEO基本情報は未判定。ただしGoogle MapsとInstagramで代替できているかを見る。
- YouTubeなし: YouTube SEOは未判定。Plan導線が弱い場合のみ「駐車場・入口動画をYouTube Shortsにも置くとよい」と提案。
- Xなし: X診断は未判定。飲食店では減点しない。

---

## 2. 判定対象メディア

| 媒体 | 判定するもの | なければ |
|---|---|---|
| 公式サイト | title、H1、地域語、業種語、メニュー、料金、アクセス、FAQ、構造化データ | 未判定 |
| Google Maps | 店舗名、カテゴリ、住所、営業時間、電話、写真、口コミ、返信、メニュー、予約URL | MVP必須。なければ診断不可に近い |
| Instagram | プロフィール、地域語、業種語、投稿文、ハイライト、地図/予約導線 | 未判定。ただしSNS導線は弱いと表示 |
| YouTube | チャンネル名、概要、動画タイトル、説明欄、地域語、アクセス/Plan動画 | 未判定 |
| TikTok | プロフィール、動画タイトル/説明、地域語、Impulse性 | 未判定 |
| X | プロフィール、地域語、最新情報、営業情報 | 未判定 |
| 外部媒体 | 食べログ、Retty、ホットペッパー等の掲載有無、表記整合 | 詳細は有料/手動確認 |

## 3. 店舗特定辞書

### 3.1 強一致シグナル

以下が2つ以上一致すれば、同一店舗の可能性が高い。

- 電話番号
- 住所
- Google Maps URL
- 公式サイトURL
- Instagramプロフィール内リンク
- 店舗名 + 市区町村

### 3.2 注意シグナル

以下がある場合は「要確認」にする。

- 同名店舗が複数ある
- 支店名が不明
- Google Mapsと公式サイトの住所が違う
- 電話番号が違う
- 旧店舗/移転前情報が残っている
- SNSプロフィールに所在地がない
- 閉業、臨時休業、移転の可能性がある

### 3.3 判定ラベル

| ラベル | 条件 |
|---|---|
| 一致 | 店舗名、住所、電話、URL、SNSのうち主要項目が整合 |
| 要確認 | 一部不一致、同名店舗、支店不明、SNSのみ曖昧 |
| 不一致 | 住所/電話/URLが明らかに別店舗 |
| 未判定 | 入力不足で確認不能 |

## 5. 公式サイト簡易SEOチェック

### 5.1 必須確認項目

| 項目 | 良い状態 | 弱い状態 |
|---|---|---|
| title | 店名 + 地域 + 業種/特徴がある | 店名だけ、または抽象語だけ |
| H1 | 店の内容が一目でわかる | ロゴ画像のみ、または不明 |
| 店舗概要 | 住所、電話、営業時間、定休日がある | 基本情報が散在/不足 |
| アクセス | 駐車場、最寄り、道順がある | 地図だけで補足なし |
| メニュー/料金 | 代表メニューと価格帯がある | 価格が見えない |
| FAQ | 初来店の不安に答えている | よくある不安が未掲載 |
| 対象客 | 誰に向いているかがわかる | 誰向けかわからない |
| 写真 | 外観、入口、店内、席、商品がある | 商品写真だけ |

### 5.2 簡易SEOスコア例

| 点数 | 状態 |
|---:|---|
| 0 | 公式サイトがない、または店舗情報がほぼない |
| 30 | 店名と基本情報はあるが、地域語/業種語/アクセスが弱い |
| 50 | 基本情報はあるが、FAQ/料金/対象客が不足 |
| 70 | 地域語、業種語、アクセス、料金、写真が概ね整っている |
| 90 | 初来店者とAIが誤解しにくい構造になっている |
| 100 | 公式サイト、Maps、SNS、口コミの情報が一貫している |

注意: 公式サイトがない場合は「0点」ではなく、媒体なしとして未判定にできる。ただし、Google MapsやInstagramで代替情報が整っていない場合は、来店前不安スコアに影響する。

## 6. Google Maps診断辞書

### 6.1 情報充足

| 項目 | 見るもの |
|---|---|
| 店舗名 | 表記が公式サイト/SNSと一致 |
| カテゴリ | 主カテゴリと副カテゴリが適切 |
| 住所 | 正確、建物名、入口補足 |
| 営業時間 | 最新、祝日、臨時休業 |
| 電話 | 通じる番号 |
| 公式URL | 正しい公式サイトまたは予約ページ |
| メニュー | 代表メニュー、価格帯 |
| 写真 | 外観、入口、駐車場、店内、席、商品 |
| 予約/注文 | 必要な業種では導線あり |

### 6.2 口コミ

| 項目 | 見るもの |
|---|---|
| 件数 | 同エリア同業種と比較 |
| 評価平均 | 数字だけでなく低評価理由を見る |
| 最新性 | 最近3か月以内に口コミがあるか |
| 返信率 | 直近30件の返信数 |
| 感情語 | 静か、親切、入りやすい、駐車場、子連れ等 |
| 公式表現との一致 | 公式で言う強みと口コミが合っているか |

### 6.3 Google Maps整備度スコア例

| 点数 | 状態 |
|---:|---|
| 0 | Maps情報がほぼ未整備、または店舗特定不可 |
| 30 | 基本情報のみ、写真/メニュー/返信が弱い |
| 50 | 営業情報はあるが、初来店不安が残る |
| 70 | 写真、口コミ、営業時間、URLが概ね整備 |
| 90 | 地図だけで初来店判断がかなりできる |
| 100 | Maps単体で見つかり、安心し、来店/予約できる |

## 7. SNS簡易SEO/GEOチェック

### 7.1 Instagram

見るもの:

- プロフィールに地域語、業種語、店の特徴があるか
- Google Maps/予約/公式サイトへの導線があるか
- ハイライトに営業時間、駐車場、メニュー、席、FAQがあるか
- 投稿文に地域語と目的語が自然に入っているか
- 保存されやすい空気/席/商品/季節の投稿があるか

### 7.2 YouTube

見るもの:

- チャンネル名と概要で地域・業種がわかるか
- 動画タイトルに地域語、業種語、目的語があるか
- 説明欄に住所、Maps、公式サイト、営業時間があるか
- 駐車場、入口、アクセス、店内紹介などPlan動画があるか

### 7.3 X

見るもの:

- プロフィールに地域・業種・営業時間があるか
- 臨時休業、空席、限定、イベントなど最新情報が機能しているか
- 公式サイト/Mapsへのリンクがあるか

### 7.4 TikTok

見るもの:

- プロフィールに地域・業種・Maps/Instagram導線があるか
- Impulseを作る短尺動画があるか
- 煽りが店の世界観を壊していないか

## 9. 未判定の扱い

| 状況 | 表示 | 総合点への扱い |
|---|---|---|
| 公式サイトなし | 公式サイトSEO: 未判定 | 直接減点しない |
| Instagramなし | Instagram診断: 未判定 | SNS導線が必要な業種では提案に回す |
| YouTubeなし | YouTube診断: 未判定 | Plan動画不足がある場合のみ改善提案 |
| Xなし | X診断: 未判定 | 原則減点しない |
| Google Mapsなし/特定不可 | 診断不可または要確認 | 重要ブロッカー |

## 10. レポート表示ルール

無料診断では、SEO/MEO/GEOを以下のように出す。

1. 店舗特定: 一致 / 要確認 / 不一致 / 未判定
2. Google Maps診断: 0-100点
3. 口コミ信頼: 0-100点
4. 簡易SEO: 判定対象がある場合のみ0-100点、なければ未判定
5. SNS簡易SEO: 媒体ごとに判定/未判定
6. 独自スコア: 改善目安としてSave/Plan/Impulse、来店前不安、世界観を表示

表示文例:

> 公式サイトSEO: 未判定  
> 公式サイトURLが未入力のため、今回は公式サイトSEOは採点していません。Google MapsとInstagram上で、営業時間・駐車場・価格帯・来店前不安が補えているかを確認しています。
